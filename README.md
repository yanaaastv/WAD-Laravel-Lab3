# Laboratory 3: Laravel Pages, Controllers, and Routes

## Project Title
**My Portfolio Project**  
*A simple multi-page portfolio application built with Laravel.*

## Project Description
This project is a basic portfolio-style web application created using Laravel. It demonstrates the use of controllers, routes, and Blade templates to manage multiple pages, including a main portfolio page and sub-pages like About and Hobbies.

## Objective

This laboratory exercise aims to help students apply the basics of Laravel installation, routing, controllers, and Blade templates by building a simple multi-page portfolio-style application.

By the end of this lab, students should be able to:

- Set up a Laravel project locally  
- Create controllers and routes  
- Render multiple Blade templates  
- Organize a basic Laravel project properly  
- Publish and document a project using GitHub

## Instructions to Run the Project Locally
Follow these steps to run the project on your local machine:

1. Clone the Repository
git clone https://github.com/yourusername/your-repo-name.git
2. Navigate to Project Folder
cd your-repo-name
3. Install Dependencies
composer install
4. Setup Environment File
cp .env.example .env
5. Generate Application Key
php artisan key:generate
6. Run the Application
php artisan serve
7. Open in Browser
http://127.0.0.1:8000
